## 映射关系

| 环境变量 | 用途 | 默认值 |
|---------|------|--------|
| `IMAGECREATOR_WORKSPACE` | 工作区目录 | `/Users/leo/Agents/imagecreator-workspace` |
| `COMFYUI_BASE_DIR` | ComfyUI 基础目录 | `/Users/leo/Documents/ComfyUI` |
| `COMFYUI_INPUT` | ComfyUI 输入目录 | `$COMFYUI_BASE_DIR/input` |
| `COMFYUI_OUTPUT` | ComfyUI 输出目录 | `$COMFYUI_BASE_DIR/output` |
| `COMFYUI_OUTPUT_ARCHIVE` | ComfyUI 输出归档 | `/Users/leo/Documents/ComfyUI_output_archive` |
| `LORA_ROOT_DIR` | LoRA 根目录 | `/Users/leo/Downloads/lora` |
| `MATERIAL_ROOT_DIR` | 材质根目录 | `/Users/leo/Documents/material` |

## 使用示例

### 设置环境变量
```bash
# Bash/Zsh
export IMAGECREATOR_WORKSPACE="/path/to/workspace"
export COMFYUI_BASE_DIR="/path/to/comfyui"
export LORA_ROOT_DIR="/path/to/lora"
export MATERIAL_ROOT_DIR="/path/to/material"

# 或在运行时设置
IMAGECREATOR_WORKSPACE="/new/path" python3 bin/tg_bot.py
COMFYUI_BASE_DIR="/opt/comfyui" python3 bin/comfyui_client.py --task task.json
```

### Docker/Compose 配置
```yaml
services:
  imagecreator:
    environment:
      - IMAGECREATOR_WORKSPACE=/workspace
      - COMFYUI_BASE_DIR=/comfyui
      - LORA_ROOT_DIR=/data/lora
      - MATERIAL_ROOT_DIR=/data/material
```

## 优势

1. **可移植性**: 代码可在不同环境运行，无需修改路径
2. **部署友好**: 支持容器化部署和云环境
3. **多用户支持**: 不同用户可使用不同的路径配置
4. **测试便利**: 测试环境可使用临时目录
5. **向后兼容**: 未设置环境变量时使用原有默认路径
