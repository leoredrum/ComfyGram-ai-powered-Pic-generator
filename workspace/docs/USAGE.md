## 基础命令

### /start

初始化机器人，开始对话。

```
/start
```

**返回**: 欢迎消息和可用命令列表

---

### /help

显示帮助信息和使用说明。

```
/help
```

**返回**: 命令列表和示例

---

### /status

查看当前任务状态。

```
/status
```

**返回**: 
- 队列中的任务数
- 当前正在执行的任务
- 最近完成的任务

---

### /cancel

取消当前正在执行的任务。

```
/cancel
```

**注意**: 只能取消未完成的任务

## 文生图（T2I）

### 基础用法

```
/t2i 一只可爱的猫咪坐在窗台上，阳光明媚，照片级真实感
```

**参数**: 
- `prompt`: 文本描述（必需）

---

### 高级参数

```
/t2i 赛博朋克城市夜景，霓虹灯，高度细节
--model flux
--steps 30
--cfg 7.5
--width 1024
--height 1024
--seed 12345
```

**参数说明**:

| 参数 | 说明 | 默认值 |
|------|------|--------|
| `--model` | 使用的模型 | flux |
| `--steps` | 生成步数 | 20 |
| `--cfg` | CFG 强度 | 7.0 |
| `--width` | 图像宽度 | 1024 |
| `--height` | 图像高度 | 1024 |
| `--seed` | 随机种子 | 随机 |
| `--batch` | 批量生成数量 | 1 |

---

### 提示词技巧

**好的提示词结构**:

```
[主体] + [细节] + [风格] + [技术参数]
```

**示例**:

```
✅ 好的提示词:
/t2i 一只金毛犬在海滩上奔跑，金色毛发在阳光下闪闪发光，
蓝海白云，电影级灯光，超写实，8k 分辨率

❌ 不好的提示词:
/t2i 狗
```

**提示词模板**:

1. **写实照片**
   ```
   /t2i [主体]，自然光照，景深效果，佳能 EOS R5 拍摄，
   85mm f/1.2 镜头，照片级真实感
   ```

2. **动漫风格**
   ```
   /t2i [主体]，新海诚风格，清新明亮，精美背景，
   动漫艺术，高质量插画
   ```

3. **艺术画作**
   ```
   /t2i [主体]，油画风格，梵高星空风格，浓墨重彩，
   艺术杰作，博物馆收藏级别
   ```

---

### 批量生成

生成多个变体：

```
/t2i 一辆红色跑车
--batch 4
```

生成不同风格：

```
/t2i 美丽风景
--batch 3
--style anime
--style realistic
--style watercolor
```

## 图生图（I2I）

### 基础用法

1. **发送图片**给机器人
2. 紧接着发送命令：

```
/i2i 将图片转换为油画风格
```

**参数**:
- 图片：通过发送图片提供（必需）
- prompt: 转换描述（必需）

---

### 图像增强

```
/i2i 增强，4k 分辨率，细节锐化
```

**用途**:
- 提高图像分辨率
- 锐化细节
- 去除噪点
- 改善光照

---

### 风格转换

```
/i2i 转换为动漫风格
```

**支持的风格**:
- 动漫 (anime)
- 油画 (oil painting)
- 水彩 (watercolor)
- 素描 (sketch)
- 赛博朋克 (cyberpunk)

---

## 风格迁移

### /style 命令

快速应用预设风格：

```
/style 动漫
/style 油画
/style 水彩
/style 素描
```

**使用方法**:
1. 发送图片
2. 发送 `/style <风格名>`

---

### 自定义风格参数

```
/style 赛博朋克
--strength 0.8
--detail_level high
```

**参数**:
- `--strength`: 风格强度 (0.0-1.0)
- `--detail_level`: 细节级别 (low/medium/high)

---

## 高级功能

### LoRA 模型

使用预配置的 LoRA：

```
/t2i mylora 一个美丽的花园
```

**查看可用 LoRA**:

```
/loras
```

**返回**: 已安装的 LoRA 列表

---

### 组合多个 LoRA

```
/t2i <主体> + lora1:0.7 + lora2:0.5
```

**格式**: `lora名称:强度`

强度范围: 0.0 - 1.0

---

### 自定义工作流

使用自定义 ComfyUI 工作流：

```
/custom my_workflow
```

**前提**: 工作流必须在 `workspace/workflows/registry.json` 中注册

---

### 图像序列

生成连续变化：

```
/t2i 一朵花
--batch 5
--interpolate true
```

**用途**: 创建动画帧或展示渐进变化

## 最佳实践

### 提示词编写

1. **具体描述**
   ```
   ✅ 一只橙色的虎斑猫，绿色眼睛，坐在木制窗台上
   ❌ 一只猫
   ```

2. **指定风格**
   ```
   ✅ 迪士尼动画风格，色彩鲜艳
   ❌ 卡通风格
   ```

3. **添加技术参数**
   ```
   ✅ 8k 分辨率，电影级灯光，景深效果
   ❌ 高质量
   ```

---

### 性能优化

1. **使用合适的分辨率**
   - 预览：512x512 或 768x768
   - 最终输出：1024x1024 或更高

2. **调整步数**
   - 快速生成：15-20 步
   - 高质量：25-30 步
   - 超高质量：40-50 步（耗时更长）

3. **批量处理**
   - 生成多个小批次而非大批次
   - 每批 2-4 张图像

---

### 常见问题

**Q: 如何提高生成速度？**

A: 
- 减少步数
- 降低分辨率
- 使用更快的模型（如 SDXL Turbo）
- 关闭不必要的后台程序

**Q: 如何获得更好的结果？**

A:
- 详细描述你想要的
- 指定风格和艺术家
- 添加技术参数
- 尝试不同的随机种子
- 使用图生图进行迭代改进

**Q: 生成结果不符合预期？**

A:
1. 检查提示词是否清晰
2. 尝试使用不同的种子
3. 使用图生图进行修改
4. 调整 CFG 参数
5. 尝试不同的模型
